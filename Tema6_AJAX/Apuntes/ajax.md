## AJAX
En AJAX, la primera petición HTTP del cliente hace que el servidor
le mande un JS, HTML y CSS generados en el backend.

Después de que la página este generada, las siguientes peticiones que se hagan 
el servidor lo que devolverá texto plano, un JSON, un XML. Es decir, no va a generar 
código, ni HTML, ni CSS, ni JS. Entonces el servidor va a funcionar como una API que manda datos.
